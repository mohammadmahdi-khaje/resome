تکنولوژی ها استفاده شده

زبان نشانه‌گذاری ابرمتنی، (به انگلیسی: Hyper Text Markup Language ) یا زنگام (به انگلیسی: HTML ) در کنار سی‌اس‌اس هستهٔ فناوری ساخت صفحه‌های وب هستند. HTML زبان توصیف ساختار صفحه‌های وب است. زبانی است برای نشانه‌گذاری ابرمتن (فرامتن) که برای تدوین قالب و طراحی صفحه‌های وب به کار می‌برند.

برای نوشتن ساختار سایت از HTML استفاده شده

CSS مخفف "Cascading Style Sheets" است. CSS یک زبان کامپیوتری برای چیدمان و ساختار صفحات وب (HTML یا XML) است. این زبان حاوی عناصر کدنویسی است و از این "سبک شیت های آبشاری" تشکیل شده است که به همان اندازه فایل های CSS نامیده می شوند.

برای زیبا کردن بدنه سایت از CSS استفاده شده

عبارت SCSS مخفف Sassy Cascading Style Sheets است. این اساساً یک نوع پیشرفته تر و تکامل یافته از زبان CSS است. ناتالی وایزنبام و کریس اپشتاین آن را خلق کردند و همپتون کتلین آن را طراحی کرد. دارای ویژگی های پیشرفته تری است که اغلب به آن Sassy CSS می گویند.

از پیش پردازنده scss و اسم گزاری BEM برای نوشتن css استفاده شده

جاوااسکریپت نوعی زبان سطح بالا، کامپایل درجا و چندالگویی است. جاوااسکریپت نحو آکولادی دارد، نوع‌دهی آن پویا است، نوع شیءگرا‌ی‌اش بر پایه پیش‌نمونه است و دارای توابع کلاس اول است. در ابتدا برندان آیک، سپس مشارکت کننده‌های کلیدی دیگری در مشخصات اکما اسکریپت اضافه شدند. سری جاوااسکریپت است

برای درست کردن slide menu از javascript استفاده شده

ری‌اکت (به انگلیسی: React )، یک کتابخانه متن‌باز جاوا اسکریپت برای ساخت رابط‌های کاربری و اجزای(Component) صفحات وب است. ری‌اکت در دو نسخه ری‌اکت جی اس (به انگلیسی: React. js ) و ری‌اکت نیتیو (به انگلیسی: React Native ) ارائه شده‌است.

react

Next. js یک فریم‌ورک رایگان و اوپن‌سورس است که می‌تواند امکانات خوبی را برای کار کردن با کتابخانه‌های ریکت به زبان جاوا اسکریپت در اختیار ما قرار دهد، البته با این تفاوت که بعد از استفاده از ریکت و ایجاد صفحات وب نیتیو، این فریم‌ورک کاری می‌کند که پردازش‌ها در سمت سرور انجام شود و صفحات به صورت استاتیک منتقل شوند.
